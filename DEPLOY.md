# Deploy to Vercel

Your code is on GitHub: [raystan17/medisync-labs-outbound-engine](https://github.com/raystan17/medisync-labs-outbound-engine)

## Deploy

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **Add New** → **Project**
3. Import **medisync-labs-outbound-engine**
4. Click **Deploy** (no config needed — `vercel.json` is already set up)

Your workspace will be live at `https://medisync-labs-outbound-engine.vercel.app` (or a custom domain)
